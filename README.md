# Clicky-Game

## Description
    This gaming application shows different images of which User selects one image by clicking. When they click, a score goes up by 1, and the screen shows the immages in random order. If User clicks the same image, the score drops back to 0. Once User's score is reset after an incorrect guess, the game restarts.
